# Changelog

All notable changes to the Vadim Light extension will be documented in this file.

## [1.0.0] - 2026-04-24

### Added
- Initial release of Vadim Light theme (IntelliJ port)
- 67 IntelliJ-style keybindings
  - Navigation bindings (back, forward, outline)
  - Editor manipulation (fold, split, word wrap)
  - Multi-cursor support (Shift+Alt+Up/Down)
  - Debug/refactoring shortcuts
- Cross-platform cmd/ctrl support (macOS/Windows/Linux)
- Semantic token colors for modern languages
- Support for VS Code 1.70.0+

### Theme Colors
- Light, readable color palette inspired by JetBrains IntelliJ
- Keywords, strings, numbers, comments properly highlighted
- Fold/unfold region colors
- Diff and merge conflict colors
- Error/warning/info indicator colors
- VCS annotation colors (added, modified, deleted lines)

### Keybindings Categories
- **Navigation:** Project explorer, outline, back/forward
- **Editing:** Cursors, line operations, fold/unfold
- **Refactoring:** Go to implementation, reveal definition, find references
- **Layout:** Split editors, maximize, zen mode, dock controls
- **Debug:** Step, hotswap, breakpoints (Java/Android)
- **Search:** Code search, file search, find/replace
